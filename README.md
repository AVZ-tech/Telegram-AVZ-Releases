# Telegram AVZ Releases

This repository contains public APK release downloads for Telegram AVZ.

Source code is not stored in this repository.

## Download

Open the **Releases** page and download the latest APK asset.

## Install

Download:

`telegram-avz-afat-release.apk`

Then install it on Android.
